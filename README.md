# Cybersecurity Labs Portfolio

This repository contains hands-on cybersecurity lab projects demonstrating SIEM monitoring, intrusion detection, and network scanning using industry-relevant tools.

##  Tools & Technologies

- Wazuh (SIEM)
- Suricata (IDS)
- Nmap (Network Scanning)
- Wireshark (Packet Analysis)
- PowerShell (Attack Simulation)

---

##  Projects Overview

### 1. Wazuh SIEM Monitoring

- Configured Wazuh for log collection and analysis
- Monitored system activity and generated alerts
- Detected suspicious behavior through log correlation

---

### 2. Suricata Intrusion Detection

- Configured Suricata IDS for network monitoring
- Analyzed network traffic for malicious patterns
- Generated alerts for suspicious activity

---

### 3. Nmap Network Scanning

- Performed host discovery and port scanning
- Identified open ports and running services
- Conducted service enumeration and network mapping

---

### 4. Attack Simulation & Detection

- Simulated attacks using PowerShell commands
- Generated detectable events (scans, suspicious activity)
- Observed and analyzed alerts in Wazuh and Suricata

---

##  Objective

The goal of these labs is to build practical, hands-on experience in cybersecurity operations, focusing on detection, monitoring, and basic threat analysis.
These projects simulate real-world security scenarios and demonstrate the ability to detect and analyze suspicious activity.
